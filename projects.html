The Simon Game

  • Implemented game logic for pattern recognition and increasing difficulty levels, alongside a best score system that persists until page refresh, ensuring a responsive and challenging user experience across different devices.
  • Integrated backend functionality using Node.js and Express.js, including a login system with user authentication. Also Connected the game to a PostgreSQL database to store and manage user credentials and high scores, ensuring data persistence and secure handling of user information.
